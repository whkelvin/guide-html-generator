1. 創建並命名新資料夾 "./out/assets/mp3/"

```
out
  |
  |__assets
      |
      |__mp3

```

2. 遵循以下格式匯入 mp3 檔案

```
./out/assets/mp3/
    |
    |__上下層科判
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__原文
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__各科範圍
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__四家合註語譯
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__師父音檔
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__本表範圍
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__default.mp3 (找不到音檔時播放的檔案)
    |
    |__簡介.mp3
    |
    |__操作方式.mp3

```

3. 遵循以下格式匯入 csv 檔案至"./assets/input/"

```
./assets/input/
    |__冊1
    |   |__index.csv
    |   |__表1.csv
    |__冊2
        |__index.csv
        |__表1.csv

```

4. 在 vscode 執行 `go run pkg/cmd/main.go`
5. 將 out 資料夾上傳至 netlify
