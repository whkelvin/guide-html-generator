1. 創建並命名新資料夾 "./out/assets/mp3/"

```
out
  |
  |__assets
      |
      |__mp3

```

2. 遵循以下格式匯入 mp3 檔案

```
./out/assets/mp3/
    |
    |__上下層科判
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__原文
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__各科範圍
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__四家合註語譯
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__師父音檔
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__本表範圍
    |   |__冊1
    |       |__表1
    |           |__音檔.mp3
    |__其他
    |   |__冊1
    |       |__音檔.mp3 (可用於放置前行開示等多餘音檔)
    |
    |__default.mp3 (找不到音檔時播放的檔案)
    |
    |__簡介.mp3
    |
    |__操作方式.mp3

```

3. 遵循以下格式匯入 csv 檔案至"./assets/input/"

```
./assets/input/
    |
    |__index.csv (格式同"根目錄範例")
    |
    |__冊1
    |   |__index.csv (格式同"冊目錄範例")
    |   |__表1.csv (格式同"表範例")
    |
    |__冊2
        |__index.csv (格式同"冊目錄範例")
        |__表1.csv (格式"同表範例")

```

4. 在 vscode 執行 `go run pkg/cmd/main.go`
5. 將 out 資料夾上傳至 netlify
